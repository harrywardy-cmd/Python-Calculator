PYTHON TKINTER CALCULATOR
=========================

A simple and fully functional calculator built in Python using Tkinter.
Designed with object-oriented programming, modular code, and clean practices.

Project Goal
------------
- Build a desktop calculator with Python and Tkinter
- Apply OOP and modular design
- Learn GUI programming and interactive input handling
- Cross-platform: Windows & macOS

Features
--------
- Basic arithmetic: +, -, *, /
- Square and square root functions
- Clear button to reset calculator
- Keyboard support for digits and operators
- Dynamic display showing current input and total expression
- Clean, modular OOP design

Dependencies
------------
- Python 3.8+
- Tkinter (built-in with Python)

Running the Calculator
----------------------
1. Clone the repository:
   git clone https://github.com/harrywardy-cmd/Python-Calculator.git
   cd python-tkinter-calculator

2. Run the app:
   python calculator.py

3. Use mouse or keyboard to enter digits and operations.

Classes & Architecture
----------------------
- Calculator Class: Handles GUI, buttons, display, and evaluation logic
- Display Labels: Show current expression and total expression
- Button Grid: Dynamically generated digits and operators
- Special buttons: C, =, x², √x

Keyboard Shortcuts
------------------
- Digits 0-9: type number keys
- Operators +, -, *, /: type respective keys
- Enter: evaluate expression
- C: clear input

Future Enhancements
-------------------
- Decimal precision handling
- Additional functions: %, negation, memory
- Button click sound effects
- UI themes and color customization
